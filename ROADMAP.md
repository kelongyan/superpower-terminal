# SuperPower Terminal 开发路线图

基于当前项目状态分析，按优先级排序的开发计划。

---

## P0 - 生产就绪核心功能（1-2 周）

### 1. 性能优化与稳定性

**优先级**: 🔴 最高  
**预计工时**: 3-5 天

- **渲染性能优化**
  - 优化 atlas 纹理分配策略，减少重建频率
  - 实现增量渲染，仅更新脏区域
  - 优化 DirectWrite 字形缓存命中率
  
- **内存管理**
  - 实现 scrollback 行数限制的自动清理
  - 优化大量输出场景下的内存占用（如 `cat large.log`）
  - 添加内存使用监控与告警

- **稳定性修复**
  - 处理 PTY 异常退出场景
  - 修复长时间运行后可能的资源泄漏
  - 增强错误恢复机制

### 2. 键盘快捷键系统

**优先级**: 🔴 最高  
**预计工时**: 2-3 天

- 实现可配置的快捷键绑定系统
- 默认快捷键集合：
  - `Ctrl+Shift+C/V`: 复制/粘贴
  - `Ctrl+T`: 新建标签页
  - `Ctrl+W`: 关闭标签页
  - `Ctrl+Tab` / `Ctrl+Shift+Tab`: 切换标签页
  - `Ctrl+F`: 搜索（为 P1 功能预留）
  - `Ctrl+Plus/Minus/0`: 字号调节/重置
- 支持在配置文件中自定义快捷键

### 3. 配置文件增强

**优先级**: 🟡 高  
**预计工时**: 2 天

- 扩展 `config.toml` 支持更多选项：
  - 快捷键绑定配置
  - 光标样式与闪烁设置
  - 滚动行为配置（平滑滚动、滚动速度）
  - 默认主题选择
- 添加配置热重载功能
- 提供配置验证与错误提示

---

## P1 - 竞争力功能（2-3 周）

### 4. 终端内搜索

**优先级**: 🟡 高  
**预计工时**: 3-4 天

- 实现 scrollback 内容全文搜索
- 支持正则表达式搜索
- 高亮所有匹配项
- 支持向上/向下跳转匹配
- 搜索 UI 集成到终端界面（类似 Ctrl+F 浮层）

### 5. 超链接支持

**优先级**: 🟡 高  
**预计工时**: 2-3 天

- 实现 OSC 8 超链接协议解析
- 自动识别 URL 模式（http/https/file 等）
- 鼠标悬停显示下划线与手型光标
- `Ctrl+Click` 打开链接
- 右键菜单支持"复制链接"

### 6. 分屏功能

**优先级**: 🟠 中高  
**预计工时**: 5-7 天

- 实现水平/垂直分屏布局
- 支持动态调整分屏比例
- 快捷键支持：
  - `Ctrl+Shift+D`: 垂直分屏
  - `Ctrl+Shift+E`: 水平分屏
  - `Alt+Arrow`: 切换焦点面板
- 每个面板独立 PTY 会话
- 分屏状态持久化

### 7. Profile 配置管理

**优先级**: 🟠 中高  
**预计工时**: 3-4 天

- 支持多 Profile 配置（开发、生产、SSH 等）
- 每个 Profile 可配置：
  - Shell 程序与参数
  - 工作目录
  - 环境变量
  - 字体与主题
  - 启动命令
- UI 支持快速切换与新建 Profile
- 配置文件支持 `[[profiles]]` 数组

---

## P2 - 体验增强（3-4 周）

### 8. 高级文本渲染

**优先级**: 🟢 中  
**预计工时**: 4-5 天

- **连字（Ligatures）支持**
  - 集成 HarfBuzz 进行文本整形
  - 支持编程字体连字（`->`, `=>`, `!=` 等）
  - 可配置开关
  
- **Emoji 与符号渲染**
  - 改进 Unicode 符号显示
  - 支持彩色 Emoji（通过 DirectWrite 或 fallback）

### 9. 命令历史与自动补全

**优先级**: 🟢 中  
**预计工时**: 3-4 天

- 记录命令历史（跨会话持久化）
- 实现命令补全提示（类似 fish shell）
- 支持历史搜索（Ctrl+R 风格）
- 智能建议基于使用频率

### 10. 主题市场与自定义

**优先级**: 🟢 中  
**预计工时**: 2-3 天

- 支持导入/导出主题配置
- 内置更多主题预设（10+ 种）
- 主题编辑器 UI（实时预览）
- 支持 iTerm2/Windows Terminal 主题格式导入

### 11. 窗口管理增强

**优先级**: 🟢 中  
**预计工时**: 2 天

- 全屏模式支持
- 窗口透明度调节
- 背景模糊效果（Windows Acrylic）
- 始终置顶选项
- 记住窗口位置与大小

---

## P3 - 长期规划（1-2 月）

### 12. 跨平台支持

**优先级**: 🔵 低  
**预计工时**: 2-3 周

- 抽象平台相关代码（PTY、字体、窗口）
- 支持 macOS（使用 Core Text）
- 支持 Linux（使用 fontconfig + FreeType）
- 统一配置文件路径处理

### 13. 扩展系统

**优先级**: 🔵 低  
**预计工时**: 2-3 周

- 设计插件 API（Lua 或 WASM）
- 支持自定义命令与快捷键
- 允许扩展修改 UI 与行为
- 提供扩展市场/仓库

### 14. 高级协议支持

**优先级**: 🔵 低  
**预计工时**: 1-2 周

- Sixel 图形协议
- Kitty 图形协议
- 同步更新模式（减少闪烁）
- 真彩色（24-bit）完整支持验证

### 15. 会话管理

**优先级**: 🔵 低  
**预计工时**: 1 周

- 保存/恢复会话状态
- 支持会话模板
- 自动重连断开的 SSH 会话
- 会话历史回放
